## Alternative Flows

### User exits the quiz
- Prompt: "Do you want to continue later?"
- Progress is saved.

### Internet connection lost
- Show offline message.
- Retry automatically when connection is restored.

### Quiz already completed
- Show today's score.
- Allow review of explanations.
