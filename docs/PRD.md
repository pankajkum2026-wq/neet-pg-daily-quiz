# Product Requirements Document (PRD)

## Product
Daily Quiz for NEET PG

## Product Vision
Build the habit of solving 10 high-quality NEET PG MCQs every day, helping students improve retention, consistency, and exam readiness through personalized daily practice.

## Problem Statement
Students preparing for NEET PG often:
- Forget previously learned topics.
- Lack a structured daily revision habit.
- Feel overwhelmed by the large syllabus.
- Don't know which subjects need more practice.

## Goal
- Increase daily active users.
- Improve revision consistency.
- Increase MCQs attempted per day.
- Improve Grand Test performance.
- Increase subscription engagement.

## Target Users
- Final-year MBBS students
- Interns
- Repeaters
- Working doctors preparing for NEET PG

## User Story
As a NEET PG aspirant, I want to solve one curated quiz every day so that I can revise consistently and improve my exam performance.

## Core Features

### 1. Daily Quiz
- 10 questions
- Mixed subjects
- Faculty-curated
- Available every day

### 2. Instant Feedback
- Correct answer
- Explanation
- Clinical pearl
- Memory trick

### 3. Results Screen
- Score
- Accuracy
- Time taken
- Weak topics
- Strong topics

### 4. Streak System
- Daily streak tracking
- Rewards for consistency

### 5. Quiz History
- Previous quizzes
- Retry incorrect questions
- Bookmark questions

### 6. Notifications
- Daily reminder
- Streak reminder
- Result summary

## Success Metrics
- Daily Quiz Completion Rate
- Daily Active Users
- Weekly Retention
- Average Streak Length
- Quiz Accuracy
- Average Time per Quiz

### Included
