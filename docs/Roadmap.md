# Product Roadmap

## Vision

Help every NEET PG aspirant build a consistent daily revision habit through engaging, personalized quizzes.

---

# Phase 1 (MVP)

## Goal
Launch a simple, reliable daily quiz experience.

### Features
- Daily Quiz (10 MCQs)
- Quiz Timer
- Instant Answer Explanations
- Results Screen
- Quiz History
- Daily Streak
- Push Notifications

**Success Criteria**
- 65% Quiz Participation Rate
- 85% Quiz Completion Rate

---

# Phase 2

## Goal
Improve engagement and learning outcomes.

### Features
- Weak Topic Analysis
- Performance Analytics
- Subject-wise Progress
- Bookmark Questions
- Retry Incorrect Questions
- Weekly Summary

**Success Criteria**
- Increase Weekly Retention by 15%
- Increase Average Streak Length

---

# Phase 3

## Goal
Deliver a personalized learning experience.

### Features
- AI-Personalized Quizzes
- Adaptive Difficulty
- Leaderboards
- Friend Challenges
- Faculty Live Quiz Events
- Smart Revision Recommendations

**Success Criteria**
- Improve Grand Test Scores
- Increase Subscription Renewal
