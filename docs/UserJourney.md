# User Journey

## Goal
Help NEET PG aspirants build a daily habit of revision through short, engaging quizzes.

## Journey

### 1. Student opens the DBMCI app
- Sees the "Today's Daily Quiz" card on the home screen.

### 2. Student starts the quiz
- Reads instructions.
- Begins a 10-question quiz.

### 3. Student attempts questions
- Selects answers.
- Receives instant feedback after each question.

### 4. Student views results
- Sees score and accuracy.
- Reviews explanations.
- Identifies weak topics.

### 5. Student earns rewards
- Streak increases.
- XP or badge is awarded for completion.

### 6. Student returns the next day
- Receives a notification for the new daily quiz.
- Continues the streak.
